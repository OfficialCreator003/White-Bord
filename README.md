A fully functional browser-based Whiteboard and Text Collaboration application built using pure **HTML, CSS, and JavaScript**.  
This project demonstrates advanced canvas drawing, state management, and interactive UI design without external frameworks.

##  Features

### Whiteboard
- Freehand drawing (Pen tool)
- Eraser tool (true canvas erase)
- Shape tools (Line, Rectangle, Circle)
- Adjustable brush size & color
- Undo / Redo system
- Clear board with confirmation
- Save whiteboard as PNG image
- Canvas-safe resizing

### Notes Editor
- Built-in text editor
- Auto-save using LocalStorage
- Word counter
- Download notes as `.txt`

### UI / UX
- Clean responsive layout
- Mode-based tool selection
- Keyboard-friendly controls
- Professional modular JavaScript architecture

##  Technologies Used
- HTML5 Canvas
- CSS3 (Responsive Layout)
- Vanilla JavaScript (ES6)

##  Use Cases
- Online teaching & explanation boards
- Brainstorming sessions
- Quick diagram drawing
- Collaborative planning tools
- Learning canvas-based web development

##  How to Run
1. Download the repository
2. Open `index.html` in any modern browser
